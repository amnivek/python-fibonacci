=========
Fibonacci
=========

This is a Python starter project. Use Fibonacci number as example.

Usage
-----

From code:

.. code-block:: python

    import fibonacci

    f = fibonacci.compute(5)

From command line:

.. code-block::

    $ fibonacci 5
    > [2020-03-03 12:34:56 +0800][fibonacci.bin][INFO] Fibonacci(5) = 5
